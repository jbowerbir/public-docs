# Privacy Policy

Last updated: March 2026

## Overview

Jake Health Tracking Automation is a personal software tool used by
Jake Bower-Bir to access health data from the Oura API for personal
tracking and analysis.

This application is not a public service and does not support external users.

## Data Access

The application retrieves the following data from the Oura API:

- Sleep metrics
- Readiness metrics
- Activity data
- Tags and related metadata

This data belongs solely to the account owner.

## Data Storage

All retrieved data is stored locally on the owner's personal systems
for personal analysis.

No centralized database or shared data storage is used.

## Data Sharing

Data retrieved through this application:

- Is **not sold**
- Is **not shared with third parties**
- Is **not distributed publicly**

Data may only be accessed by the owner of the application.

## Security

Authentication with the Oura API is handled through the official OAuth
authorization process provided by Oura.

API credentials and tokens are stored locally and are not committed to
public repositories.
